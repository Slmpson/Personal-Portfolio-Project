Space Invaders

A JavaScript Game

Skills Used:

CSS

JavaScript

Based on the well known arcade game Space Invaders, this project was adapted from a YouTube video by Ania Kubow on the freeCodeCamp.org channel. The video can be found here.

My version of Space Invaders starts with an intro screen displaying the controls for the game and a start button. 1 point is gained for each enemy destroyed. After 30 enemies have been destroyed the level is finished and the next level can be commenced, with the enemies gradually gaining speed each level.
