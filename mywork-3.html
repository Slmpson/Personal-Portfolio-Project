Space Invaders

A JavaScript Game

Skills Used:

CSS

JavaScript

Based on the well known arcade game Space Invaders, this project was adapted from a YouTube video by Ania Kubow on the freeCodeCamp.org channel. The video can be found here.

This version of Space Invaders starts with an intro screen displaying the controls for the game and a start button. Points are accumulated for each kill. Once all enemies are destroyed the level is complete and a next level option appears. Enemies gather speed with each level.
