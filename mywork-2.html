A React application

A Pokedex interface created using CSS, React and the Pokemon API. This project was adapted from a YouTube video by Web Dev Simplified found here.

My pokedex application shows the first 10 pokemon on the left which can be individually selected to show information within the screen on the right hand side. The previous and next buttons display the following 10 pokemon on the list. Alternatively, the search bar can be used to display the same information. A pokemon can be searched by name or number.

The D-Pad is made from individual buttons which cycles through different information. The left and right arrow moves to the previous or next pokemon, whereas the up and down button cycle through information on the selected pokemon.
